HowTo — 学生向け Quick Start（IT／Linux 初心者向け）

このページは IT や Linux に不慣れな学生さんが、最短でこのシミュレータを起動して学べるように書かれた簡易手順です。

必要なもの
- モダンなブラウザ（Chrome / Edge / Firefox 推奨）
- （任意）Python 3：ローカルサーバで安定動作させたい場合に便利

1) まず試す（とても簡単）
- リポジトリ直下の `index.html` をダブルクリックして、ブラウザで開いてください。
- ページが表示されれば、そのまま操作を始められます。

2) 推奨：ローカルサーバで開く（安定）
- ターミナル（PowerShell）を開き、リポジトリのルートで次を実行します:

```powershell
python -m http.server 8000
```

- ブラウザで次を開く:
http://localhost:8000/

3) シミュレータの簡単な操作手順
- 「インストールメディアを挿入」をクリック
- ブートメニューで「Install AlmaLinux」を選ぶ
- 言語は「日本語」で進めて問題ありません
- インストール概要画面で順に設定（必須: ユーザー名、パスワード）
  - パスワードは画面の要件に従ってください（例: 8 文字以上、英数字混在など）。実際のパスワードは保存されません。
- 「インストール開始」を押すとフェイクログとプログレスバーが流れます
- 完了画面でログイン画面へ進みます。作成したユーザーでログインするか、講師が指示したデモアカウントが使える場合はそれで試してください

4) 進捗の保存
- 完了画面で「進捗を保存（JSON ダウンロード）」を押すと `progress.json` がダウンロードされます
- `progress.json` は到達したステップのみを記録し、パスワードなどの機密情報は含みません

5) よくあるトラブル（初心者向け）
- ページが表示されない: 別のブラウザを試すか、手順 2 のローカルサーバを試してください
- サーバを立てたのに見えない: ブラウザで `http://localhost:8000/` を直接入力してみてください
- エラーの確認: ブラウザで F12 を押して「Console」タブを確認してください
- 解決しない場合は担当の教員に相談してください

まずは `index.html` を開いて、画面の指示に従って進めてみてください。学習が始めやすいように最小限の手順にまとめています。
