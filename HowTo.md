HowTo — 学生向け / 講師向け（簡易まとめ）

## 学生向け（Quick Start）

- 最も簡単: リポジトリ直下の `index.html` をブラウザで開く（ダブルクリックで可）。
- ローカルサーバ推奨（安定動作）:

```powershell
python -m http.server 8000
# ブラウザで開く: http://localhost:8000/
```

- 操作の流れ（学生が行う）:
  1. 「インストールメディアを挿入」→ 次へ
  2. ブートメニューで “Install AlmaLinux” を選択
  3. 言語は日本語で進めてよい
  4. インストール概要（インストール先 / ソフトウェア / ユーザー作成）を順に入力
  5. ユーザー作成ではパスワードバリデーションに従う（パスワードは保存されません）
  6. 「インストール開始」を押すとフェイクログとプログレスバーが表示されます

- 進捗保存: 完了画面で「進捗を保存（JSON ダウンロード）」を押して `progress.json` を取得してください。

## 講師向け（運用の手引き）

- 教室準備:
  - 受講者用PCにモダンブラウザ（Chrome/Edge/Firefox）を用意してください。
  - インターネット不要だが、ローカルファイルを開けること。大人数ではローカルサーバ配信推奨。

- 授業の流れ（例）:
  1. 目的と目標の説明（約5分）
  2. デモ（講師が `index.html` を表示）
  3. 学生が個別に演習（20〜40分）
  4. 学生が `progress.json` をダウンロードして提出
  5. 講師は `teacher.html` を使って複数の `progress.json` を集計、CSV出力

- 進捗集約（`teacher.html` の使い方）:
  - `teacher.html` を開き、回収した `progress.json` を複数選択してアップロードします。
  - 「集計」→ CSV ダウンロード（ブラウザ内完結、外部へ送信されません）。

## よくあるトラブルと対処

- ページが表示されない: ブラウザのコンソールを確認し、依存ファイルが同一フォルダにあるか確認してください。
- JSON がダウンロードできない: ブラウザのダウンロード権限やポップアップ設定を確認してください。

## 注意事項

- セキュリティ: `progress.json` は操作到達履歴のみを含み、パスワード等の機密情報は含みません。
- このシミュレーターは学習用の疑似体験です。実際のディスク操作や OS インストールは行いません。
